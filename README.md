# C Length Converter

This project is one of the first console applications I wrote while learning the C programming language.

The program takes a length value in centimeters (cm) from the user and converts it to meters (m) and kilometers (km), then prints the results to the console.

## Features
- Console-based C application
- Converts centimeters to meters and kilometers
- Simple and easy-to-understand code
- Uses basic C programming concepts

## Technologies Used
- C Programming Language
- Standard C Library (`stdio.h`)

Note:
The .exe file is compiled from source code.

This may trigger a security warning.

# C Uzunluk Dönüştürücü

Bu proje, C programlama dilini öğrenirken yazdığım ilk konsol uygulamalarından biridir.

Program, kullanıcıdan santimetre (cm) cinsinden bir uzunluk değeri alır ve bunu metre (m) ve kilometre (km) cinsine dönüştürür, ardından sonuçları konsola yazdırır.

## Özellikler
- Konsol tabanlı C uygulaması
- Santimetreleri metre ve kilometreye dönüştürür
- Basit ve anlaşılması kolay kod
- Temel C programlama kavramlarını kullanır

## Kullanılan Teknolojiler
- C Programlama Dili
- Standart C Kütüphanesi (`stdio.h`)

.exe dosyası kaynak koddan derlenmiştir.
