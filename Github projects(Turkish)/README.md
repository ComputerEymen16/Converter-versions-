# C Uzunluk Dönüştürücü

Bu proje, C programlama dilini öğrenirken yazdığım ilk konsol uygulamalarından biridir.

Program, kullanıcıdan santimetre (cm) cinsinden bir uzunluk değeri alır ve bunu
metre (m) ve kilometre (km) cinsine dönüştürür, ardından sonuçları konsola yazdırır.

## Özellikler
- Konsol tabanlı C uygulaması
- Santimetreyi metre ve kilometreye dönüştürür
- Basit ve anlaşılması kolay kod
- Temel C programlama kavramlarını kullanır

## Kullanılan Teknolojiler
- C Programlama Dili
- Standart C Kütüphanesi (`stdio.h`)

Not:
.exe dosyası kaynak koddan derlenmiştir.

Bu bir güvenlik uyarısına neden olabilir.
